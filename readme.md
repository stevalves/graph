## Descrição do Projeto

Este projeto implementa análise e processamento de matrizes baseadas em um dataset de preferências de categorias de anime da turma. Os objetivos incluem:

1. Geração de três tipos de matrizes a partir do dataset
2. Construção de grafos representando as relações entre dados
3. Cálculo de métricas topológicas para análise estrutural
4. Documentação completa do processo e resultados

### Dataset

**Fonte:** Categorias de anime preferidas dos alunos da turma  
**Tipo de Análise:** Matriz de incidência, similaridade e coocorrência

## Pré-requisitos

- Python 3.8 ou superior
- pip (gerenciador de pacotes Python)

## Instruções de Configuração e Execução

### 1. Criar e Ativar o Ambiente Virtual

Para Windows (PowerShell):

```powershell
python -m venv .venv
.\.venv\Scripts\activate
```

### 2. Instalar Dependências

Com o ambiente virtual ativado, instale as dependências listadas em `requirements.txt`:

```powershell
pip install -r requirements.txt
```

### 3. Executar o Programa Principal

Com o ambiente virtual ativado, execute:

```powershell
python main.py
```
